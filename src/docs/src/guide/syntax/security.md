## 禁用公式显示

计算书中有些代码只用于组织数据或调用外部资源，不适合作为公式展示。可以使用隐藏或禁用相关能力控制输出。

常见做法是使用 `hide()` 和 `show()` 包裹不需要展示的代码：

```python
hide()
temporary_value = 1 + 2
show()

"后续正文继续正常输出"
```

## 文档加密

UzonCalc 文档站示例中包含 VuePress 的页面加密能力，适合保护文档站页面内容。计算报告本身如需权限控制，建议在部署环境或文件分发流程中处理。

## 编写建议

- 关键计算过程应尽量保留，便于审查。
- 数据准备、图表配置、外部资源路径等辅助代码可以隐藏。
- 涉及敏感信息时，不要把密钥、账号、内部地址直接写入计算书。
