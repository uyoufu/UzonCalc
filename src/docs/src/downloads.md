## 1.4.0

> 更新日期：2026-07-21

### 功能新增

1. 公式显示支持上标与分组操作数，可使用质数字符（′）作为上标
2. 块级 LaTeX 公式使用 KaTeX 渲染，显示更美观专业
3. 比较运算符（≥、≤、≠ 等）自动转换为数学符号
4. 新增粗体、斜体及红/绿/黄色彩样式助手，丰富文本表现力
5. 二维数组自动渲染为矩阵表格
6. 新增 `uzoncalc zip` 命令，可将脚本打包为 .uzc 归档文件，并支持 shebang 直接执行
7. 新增 Figure 助手，支持插入 SVG 图形
8. 新增工作区多标签页、就地执行和分离式依赖编辑器
9. .uzc 归档可嵌入 PNG 缩略图，便于预览与分享

### 功能优化

1. 新增公式表达式渲染开关，可灵活控制公式显示
2. 表达式结果可直接捕获复用，避免重复计算
3. 分享链接持久化保存，刷新页面后分享状态不丢失
4. 新增收藏夹和发布菜单，报告管理更便捷
5. 页面整体重新设计，分享与管理功能体验更完善
6. 国际化和翻译文本优化，显示更清晰准确
7. 核心依赖拆分为可选安装，减小基础安装体积

### Bug 修复

1. 修复布尔值在手算输出中无法正确渲染的问题
2. 修复纯变量后脚本符号匹配失败的问题
3. 修复表格单元格中单位被错误重排的问题
4. 修复顶层 await 调用被错误记录到公式中的问题
5. 修复 .uzc 打包时的类型兼容性问题
